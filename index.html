Занимался разработкой собственных веб приложений, по большей части на Flash/AS3. Реализовывал их взаимодействие с веб сервером (Apache, php + MySQL). Подключал к проектам СМС биллинги. Имел небольшой опыт размещения приложений в соцсетях вконтакте и фейсбуке и взаимодействия с их API. Так же работал с пиринговой сетью Flash P2P для создания реалтайм мультиплеерных игр без использования отдельного сервера. Знакомился с мультиплеерным сервисом player.io. Использовал физические движки BOX2D, NAPE, графические Starling, Alternativa 3D. Еще занимался созданием сайтов (html, CSS, JavaScript (+AJAX), php, MySQL), делал свои движки + использовал готовые CMS.
